NgeShare - Membuat Wifi Sendiri di Xubuntu 17.04

Jun 26, 2017
#tutorial #linux


Beberapa hari yang lalu saya baru saja mencoba Xubuntu 17.04 yang sebenarnya telah rilis sekitar dua bulan yang lalu, yaitu pada bulan April. Mungkin ini kesannya kudet, alias kurang kurang update, tapi ya maklum saja karena dulunya saya masih terlalu nyaman dengan versi sebelumnya, yaitu versi 16.04 yang sudah saya custom di sana-sini. Namun, seiring dengan berjalannya waktu, ketidaknyamanan mulai menghampiri dan memaksa saya untuk akhirnya move on ke versi terbarunya. Berdasarkan rekomendasi dari kakak saya yang juga seorang pengguna Linux, tetapi ia lebih expert, akhirnya saya benar-benar beralih ke versi terbarunya dari Xubuntu.

Well, setelah menginstall dan menggunakan Xubuntu versi terbaru ini, ya wajarlah, saya mendapati beberapa pembaruan di dalamnya, mulai dari user interfacenya, kernel Linux 4.10, hingga hadirnya swapfile. Tapi dari sekian banyak pembaruan yang ada di versi terbarunya ini, terdapat satu hal baru yang sangat menarik perhatian saya. Hal baru itu adalah fitur wifi yang memungkinkan pengguna untuk membuat sinyal wifi sendiri dari modem yang digunakannya. Sebenarnya fitur ini sudah dapat digunakan pada versi sebelumnya, akan tetapi untuk memunculkannya harus melalui langkah-langkah yang sangat njelimet (ribet), terlebih untuk pemula seperti saya ini.

Pada Xubuntu 17.04 ini, jika pengguna ingin mengubah modemnya menjadi wifi, dapat dengan mudah melakukannya. Bagaimana cara untuk menggunakannya? Oke, perhatikan langkah-langkah yang akan saya berikan berikut ini.
1. Langkah pertama yang harus dilakukan, yaitu pastikan modem yang akan digunakan telah terpasang dan terinstall.
2. Jika modem telah terpasang dan terinstall, lalu install dnsmasq dengan memasukkan perintah di bawah ini ke terminal.
sudo apt install dnsmasq

3. Ketika proses install selesai, masukkan perintah berikut ini.
sudo nano /etc/dnsmasq.conf

4. Setelah memasukkan perintah di atas ke dalam terminal, akan muncul tampilan seperti di bawah ini.

5. Pada tampilan di atas masukkan kode dhcp-range=10.10.0.20,10.10.0.30,12h di bagian paling atas seperti pada tampilan di bawah ini. Kemudian simpan dan keluar dari tampilan ini.

6. Langkah selanjutnya adalah dengan membuka fitur Edit Connections....
7. Di fitur tersebut, klik tombol Add.

8. Lalu akan muncul tampilan seperti berikut ini.

9. Pada tampilan tersebut, klik pilihan Ethernet dan ganti dengan Wifi, lalu klik tombol Create.

10. Di langkah berikutnya akan muncul tampilan jendela Editing Wi-Fi connection 1.

11. Di jendela tersebut terdapat pilihan Mode yang menujukkan Client. Nah, Client tersebut kamu ganti dengan Hotspot.

12. Kemudian isi SSID dan ganti Connection name dengan nama wifi yang diinginkan. Misalnya kita menggunakan nama wifi sinyal wifi.

13. Usai langkah di atas, pilih menu General. Pada menu ini, hilangkanlah ceklist di samping kiri tulisan Automatically connect to this network when it is available.


14. Sebelum kamu menge-klik tombol Save, jangan lupa untuk membuat password wifimu dengan memilih menu Wi-Fi Security.

15. Pada pilihan Security, ganti None menjadi WPA & WPA2 Personal. Lalu isi kolom password dengan password yang diinginkan.


16. Setelah membuat password, baru kamu dapat menyimpannya dengan menge-klik tombol Save.

Ketika telah menyelesaikan enam belas langkah di atas, kamu belum dapat langsung menggunakan wifimu.
Lalu bagaimana cara untuk menggunakannya?

Oke, untuk menggunakannya, pertama kamu pilih menu Connect to Hidden Wi-Fi Network.... Kemudian akan muncul tampilan jendela seperti di bawah ini.

Pada pilihan Connection, kamu tekan dan pilih wifi yang telah kamu buat tadi. Lalu klik tombol Connect.

Ketika telah muncul notifikasi seperti di bawah ini, maka wifi siap untuk digunakan.

Nah, itu dia langkah-langkah untuk membuat wifi di Xubuntu 17.04. Semoga beberapa langkah atau lebih tepatnya tutorial ini dapat bermanfaat. Jika sekiranya ada yang perlu untuk ditanyaka, maka kamu bisa menanyakannya di kolom komentar di bawah ini. Salam linux lover…😁
