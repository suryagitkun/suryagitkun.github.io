NgeShare - Cara Copy Paste ke File System di Lubuntu 18.04

Oct 7, 2018
#tutorial #linux


Ketika sedang menggunakan sistem operasi Linux Lubuntu 18.04, apakah kamu pernah mencoba melakukan tindakan seperti pada tampilan di bawah ini?



Jika iya, maka wajar saja bila hal tersebut tidak dapat dilakukan karena pemindahan file pada direktori ‘File System’ memerlukan akses ‘root’ atau dikenal juga dengan istilah ‘superuser’. Terkait dengan hal ini, pada kesempatan kali ini saya akan membagikan tips agar kamu dapat melakukan tindakan tersebut pada direktori 'File System'. Baiklah, tanpa perlu panjang lebar, mari simak tips berikut ini.
1. Buka ‘Terminal’ (CTRL+ALT+T).
2. Masukkan perintah ‘sudo su’ pada terminal untuk masuk menuju root.

3. Ketika kamu telah masuk sebagai ‘root’, ketikkan perintah ‘nautilus’.

4. Setelah mengetikkan ‘nautilus’, akan muncul tampilan direktori seperti di bawah ini. Pada tampilan ini, kamu pilih direktori 'File System'. Nah, di sini, kamu dapat meletakkan/ mem-paste-kan file maupun folder yang telah dicopy/ dicut tadi.


Untuk keluar dari ‘root’, kamu klik tombol ‘close window’ pada jendela direktori tersebut terlebih dahulu.

Kemudian ketikkan perintah ‘exit’ pada terminal.

Oke, itu sekiranya tips yang dapat saya bagikan hari ini. Bila sekiranya ada yang belum dimengerti atau memiliki saran, pastinya kamu dapat menanyakan maupun menuliskannya pada kolom komentar, ya. Semoga tips ini bermanfaat. 😁
