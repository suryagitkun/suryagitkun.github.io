NgeShare - Install Google Earth 7 di Ubuntu 12.10/12.04/ Linux MInt 13

Nov 1, 2012
#tutorial #linux


Dalam tutorial ini saya akan membantu anda menginstal Google Earth 7 di Ubuntu / Linux Mint. Versi terbaru ini mencakup fitur-fitur baru yang lebih menarik seperti citra 3D dan pemandu wisata. Fitur-fitur ini memungkinkan pengguna untuk melihat beberapa kota di 3D yang akan menjadi luar biasa bagi pemilik HDTV dan monitor layar lebar. Untuk informasi lebih lanjut tentang rilis terbaru, klik di sini.
Proses instalasi untuk distribusi Linux berikut:
  • Ubuntu 12.10/12.04/11.10
  • Linux Mint 13/12
  • Debian 6.0 Squeeze

    • Instal Google Earth 7
    Dalam sistem 32-bit, masukan perintah-perintah ini di terminal:
    sudo apt-get install lsb-core

    cd /tmp

    wget -c http://goo.gl/YEqTf -O google-earth-stable_i386.deb

    sudo dpkg -i google-earth-stable_i386.deb

    Untuk sistem 64-bit, jalankan perintah ini di terminal:
    sudo apt-get install lsb-core

    cd /tmp

    wget -c http://goo.gl/t6E3i -O google-earth-stable_amd64.deb

    sudo dpkg -i google-earth-stable_amd64.deb

    Anda juga dapat men-download Google Earth dari situsnya di sini.
